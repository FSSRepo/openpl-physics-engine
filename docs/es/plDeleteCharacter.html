Documentacion de la API
plDeleteCharacter

Eliminar un character o personaje.

void plDeleteCharacter(Pluint chr)

Parametros:

chr

La id del character a eliminar.

Errores:

PL_INVALID_NAME es generado si el character esta enlazado o esta siendo usado. Tambien puede ser causado por que el character que tratas de eliminar ya no existe.
