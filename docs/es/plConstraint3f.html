Documentacion de la API
plConstraint3f

Poner un valor a un parametro especifico

void plConstraint3f(Plenum param, Plfloat x, Plfloat y, Plfloat z)

Parametros:

param

Especifica el parametro para introducir el valor. Puede ser PL_CONSTR_PIVOT_A, PL_CONSTR_PIVOT_B, PL_CONSTR_IN_AXIS_A, PL_CONSTR_IN_AXIS_B, PL_CTWIST_LIMITS, PL_HC_LIMITS, PL_HC2_LIMITS, PL_G6D_LOWER_LIN_LIMIT, PL_G6D_UPPER_LIN_LIMIT, PL_G6D_LOWER_ANG_LIMIT, PL_G6D_UPPER_ANG_LIMIT, PL_G6D_LIMIT_LIN_X, PL_G6D_LIMIT_LIN_Y, PL_G6D_LIMIT_LIN_Z, PL_G6D_LIMIT_ANG_X, PL_G6D_LIMIT_ANG_Y, PL_G6D_LIMIT_ANG_Z, PL_G6DS_STIFFNESS_SPRING, PL_G6DS_DAMPING_SPRING, PL_G6DS_EQUILI_POINT_SPRING, PL_G6DS2_LOWER_LIN_LIMIT, PL_G6DS2_UPPER_LIN_LIMIT, PL_G6DS2_LOWER_ANG_LIMIT, PL_G6DS2_UPPER_ANG_LIMIT, PL_G6DS2_STIFFNESS_SPRING, PL_G6DS2_DAMPING_SPRING, PL_G6DS2_EQUILI_POINT_SPRING, PL_G6DS2_TARGET_VELOCITY, PL_G6DS2_SERVO_TARGET, PL_G6DS2_MAX_MOTOR_FORCE, PL_G6DS2_BOUNCE, PL_P2P_PIVOT_A, PL_P2P_PIVOT_B.

x y z

Los valores asignados.

Notas:

Las siguientes constantes son aceptadas por param:

PL_CONSTR_PIVOT_A poner pivote A

PL_CONSTR_PIVOT_B poner pivote B

PL_CONSTR_IN_AXIS_A poner axis A

PL_CONSTR_IN_AXIS_B poner axis B
