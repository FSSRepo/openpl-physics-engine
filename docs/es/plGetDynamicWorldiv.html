Documentacion de la API
plGetDynamicWorldiv

Obtener un valor a traves de un parametro especifico.

void plGetDynamicWorldiv(Plenum param, Plint* values, Plsizei lenght)

Parametros:

param

Especifica el parametro para obtener los valores. Puede ser PL_RAY_BODY_RESULT.

values

El arreglo de valores

lenght

Longitud del arreglo.

Notas:

Las siguientes constantes son aceptadas por param:

PL_RAY_BODY_RESULT obtener las id de los cuerpos rigidos que fueron golpeados por el ray casting.

Errores:

PL_INVALID_OPERATION es generado si se intenta obtener el parametro PL_RAY_BODY_RESULT sin antes haber hecho una prueba de raycasting.

PL_INVALID_ENUM es generado si param es una constante invalida o no permitida.

PL_INVALID_VALUE es generado si lenght es 0 o incorrecto.
