Documentacion de la API
plVehiclefv

Poner un valor a un parametro especifico

void plVehiclefv(Plenum param, Plfloat* values, Plsizei lenght)

Parametros:

param

Especifica el parametro para introducir el valor. Puede ser PL_ADD_WHEEL.

values

Los valores asignados.

lenght

La longitud del arreglo de valores.

Notas:

Las siguientes constantes son aceptadas por param:

PL_ADD_WHEEL agregar informacion de una rueda. El arreglo debe tener la siguiente estructura de datos:

Offset Tipo Descripcion
[0...2] vector3 Punto de conexion con el chasis
[3...5] vector3 Direccion de la llanta
[7...9] vector3 Eje de giratorio de la llanta
[10] float radio de la llanta
[11] float Es una llanta frontal [0:si,1:no]

lenght debe ser 12

Errores:

PL_INVALID_NAME es generado si el cuerpo rigido no existe o el id es incorrecto.

PL_INVALID_OPERATION es generado en raras ocaciones.

PL_INVALID_ENUM es generado si param es una constante invalida o no permitida.
