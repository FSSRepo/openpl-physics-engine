Documentacion de la API
plGetCharacterfv

Obtener un valor a un parametro especifico

void plGetCharacterfv(Pluint indx, Plenum param, Plfloat* values, Plsizei lenght)

Parametros:

indx

ID del personaje

param

Especifica el parametro para obtener el valor. Puede ser PL_TRANSFORM,

values

Valores para obtener.

lenght

La longitud del arreglo de valores.

Notas:

Las siguientes constantes son aceptadas por param:

PL_TRANSFORM obtener la transformacion del personaje. values es una Matrix4x4. lenght debe ser 16

Errores:

PL_INVALID_NAME es generado si el cuerpo rigido no existe o el id es incorrecto.
