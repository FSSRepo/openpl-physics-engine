Documentacion de la API
plGetWheelfv

Obtener un valor a un parametro especifico

void plGetWheelfv(Plint wheel, Plenum param, Plfloat* values, Plsizei lenght)

Parametros:

wheel

El index de la llanta

param

Especifica el parametro para obtener el valor. Puede ser PL_WHEEL_TRANSFORM, PL_WHEEL_TRANSFORM_INTERPOLATION.

values

Valores para obtener.

lenght

La longitud del arreglo de valores.

Notas:

Las siguientes constantes son aceptadas por param:

PL_WHEEL_TRANSFORM obtener la transformacion de la llanta. values es un matrix4x4. lenght debe ser 16

PL_WHEEL_TRANSFORM_INTERPOLATION obtener la transformacion de la llanta. (interpolado)* values es un matrix4x4. lenght debe ser 16

Errores:

PL_INVALID_NAME es generado si el cuerpo rigido no existe o el id es incorrecto.

PL_INVALID_OPERATION es generado en raras ocaciones.

PL_INVALID_ENUM es generado si param es una constante invalida o no permitida.
