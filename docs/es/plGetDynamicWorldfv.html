Documentacion de la API
plGetDynamicWorldfv

Obtener un valor a traves de un parametro especifico.

void plGetDynamicWorldfv(Plenum param, Plfloat* values, Plsizei lenght)

Parametros:

param

Especifica el parametro para obtener el valor. Puede ser PL_RAY_POINT_RESULT, PL_RAY_NORMAL_RESULT.

values

El arreglo de valores

lenght

Longitud del arreglo.

Notas:

Las siguientes constantes son aceptadas por param:

PL_RAY_POINT_RESULT obtener los puntos de golpe del raycasting.

PL_RAY_NORMAL_RESULT obtener las normales del raycasting.

Errores:

PL_INVALID_OPERATION es generado si se intenta obtener el parametro PL_RAY_POINT_RESULT o PL_RAY_NORMAL_RESULT sin antes haber hecho una prueba de raycasting.

PL_INVALID_ENUM es generado si param es una constante invalida o no permitida.

PL_INVALID_VALUE es generado si lenght es 0 o incorrecto.
