Documentacion de la API
plDeleteConstraint

Eliminar una restriccion.

void plDeleteConstraint(Pluint ctr)

Parametros:

ctr

La id de la restriccion a eliminar.

Errores:

PL_INVALID_NAME es generado si la restriccion esta enlazado o esta siendo usado. Tambien puede ser causado por que la restriccion que tratas de eliminar ya no existe.
