Documentacion de la API
plConstraintfv

Poner un valor a un parametro especifico

void plConstraintfv(Plenum param, Plfloat* values, Plsizei lenght)

Parametros:

param

Especifica el parametro para introducir el valor. Puede ser PL_CONSTR_FRAME_A, PL_CONSTR_FRAME_B.

values

Los valores asignados.

lenght

La longitud del arreglo de valores.

Notas:

Las siguientes constantes son aceptadas por param:

PL_CONSTR_FRAME_A poner transformacion. matrix4x4lenght debe ser 16

PL_CONSTR_FRAME_B poner transformacion. matrix4x4 lenght debe ser 16

Errores:

PL_INVALID_NAME es generado si el cuerpo rigido no existe o el id es incorrecto.

PL_INVALID_ENUM es generado si param es una constante invalida o no permitida.
