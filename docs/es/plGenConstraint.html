Documentacion de la API
plGenConstraint

Generar un cuerpo rigido.

Pluint plGenConstraint(Plint type)

Parametros

type

Especificar el tipo de restriccion a crear.

Notas:

Tipos de restricciones:

Tipo Constante
Bisagra PL_HINGE_CONSTRAINT o PL_HINGE2_CONSTRAINT
Deslizante PL_SLIDER_CONSTRAINT
Union de 2 puntos PL_POINT2PONT_CONSTRAINT
6 grados de libertad PL_G6DOF_CONSTRAINT
6 grados de libertad tipo puente PL_G6DOF_SPRING_CONSTRAINT o PL_G6DOF_SPRING2_CONSTRAINT.
Rag Dolls de huesos PL_CONE_TWIST_CONSTRAINT

Esta funcion retorna una id y crea una cola de variables necesarias antes de crear una instancia del objeto.

Errores:

PL_INVALID_OPERATION es generado si hay una cola pendiente para crear un cuerpo rigido.
