Muy facil de usar

Open PL es un proyecto open source desarrollado por Fast Smart Sytem, trata de un motor de fisicas muy facil de implementar y usar, basado en Bullet Physics para hacer las simulaciones. El proyecto fue basado en la API Open GL por tanto, si ya lo dominas, no te preocupes, estas solo a un paso de saber usar esta API.

Muy facil de implementar

La implementacion fue hecho lo menos rigurosa posible, solo con descargar y configurar la API en tu proyecto, basta.

Se basa en Bullet Physics 2.87

Tiene como nucleo la version mas reciente del motor de fisicas mas conocido, por tanto, estas ante uno de los mejores softwares de simulacion de fisicas.
