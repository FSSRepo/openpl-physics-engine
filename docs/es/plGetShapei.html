Documentacion de la API
plGetShapei

Obtener un valor a un parametro especifico

void plGetShapei(Pluint shape, Plenum param, Plint* value)

Parametros:

param

Especifica el parametro para introducir el valor. Puede ser PL_SHAPE_TYPE, PL_USER_POINTER, PL_NUM_CHILD_SHAPES.

value

El valor asignado.

Notas:

Las siguientes constantes son aceptadas por param:

PL_SHAPE_TYPE obtener el tipo de forma de colision.

PL_USER_POINTER obtener el user pointer.

PL_NUM_CHILD_SHAPES obtener el numero de children. Debe ser compound shape.

Errores:

PL_INVALID_NAME es generado si la forma de colision no existe o el id es incorrecto.

PL_INVALID_ENUM es generado si param es una constante invalida o no permitida.
