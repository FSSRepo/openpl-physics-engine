Documentacion de la API
plDynamicWorld3f

Poner un valor a un parametro especifico

void plDynamicWorld3f(Plenum param, Plfloat v0, Plfloat v1, Plfloat v2)

Parametros:

param

Especifica el parametro para introducir el valor. Puede ser PL_GRAVITY, PL_USE_AXIS_SWEEP3.

v0 v1 v2

Valores asignados.

Notas:

Las siguientes constantes son aceptadas por param:

PL_GRAVITY poner el vector de gravedad, por defecto es (0,-9.8,0). v0 v1 v2 deben ser el vector de gravedad (x,y,x).

PL_USE_AXIS_SWEEP3 usar AxisSweep3 v0 v1 v2 deben ser el vector de extension.

Errores:

PL_INVALID_ENUM es generado si el parametro especificado no es permitido.
