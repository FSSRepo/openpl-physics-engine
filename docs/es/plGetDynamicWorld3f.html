Documentacion de la API
plGetDynamicWorld3f

Obtener un valor a traves de un parametro especifico.

void plGetDynamicWorldi(Plenum param, Plfloat* v0, Plfloat* v1, Plfloat* v2)

Parametros:

param

Especifica el parametro para obtener los valores. Puede ser PL_GRAVITY, PL_RAY_POINT_RESULT.

v0 v1 v2

Los valores para obtener.

Notas:

Las siguientes constantes son aceptadas por param:

PL_GRAVITY obtener el vector de gravedad en el orden (x,y,z).

PL_RAY_POINT_RESULT obtener el punto de golpe del ray casting (x,y,z).

Errores:

PL_INVALID_ENUM es generado si param es una constante invalida o no permitida.
