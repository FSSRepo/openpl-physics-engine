Lenguajes soportados

La API funciona de forma nativa gracias a C++, por tanto el codigo ejecutado esta acelerado por hardware. Aunque tambien cuenta con un backend en Java por tanto puede ser usado para multiples proyectos:

  • Java
  • C++
    • Plataformas y Modo de ejecucion

    Es soportado por todos los sistemas operativos, siempre y cuando el codigo del proyecto este compilado en la aquitectura y ejecutable correspondiente:

    Plataforma Nativo (C++) Backend (Java) Arquitectura
    Android Si Si arm-v7, arm64, x86, x86_64
    Windows Si Si x86, amd64
    Linux Si Si x86, amd64
    Es muy estable

    Ha sido probado en multiples ejemplos, proyectos, resulta que es algo muy innovador y eficiente, ademas para las personas que empiezan en el mundo de la programacion de videojuegos les sera muy facil implementar las fisicas sin la necesidad de comenzar de 0.
