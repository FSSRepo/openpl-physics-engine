Documentacion de la API
plConstraintf

Poner un valor a un parametro especifico

void plConstraintf(Plenum param, Plfloat value)

Parametros:

param

Especifica el parametro para introducir el valor. Puede ser PL_HC_MAX_MOTOR_IMPULSE, PL_HC_TARGET_VELOCITY, PL_SLDR_SOFTNESS_DIR_LINEAR, PL_SLDR_SOFTNESS_DIR_ANGULAR, PL_SLDR_SOFTNESS_LIMIT_LIN, PL_SLDR_SOFTNESS_ORTHO_LIN, PL_SLDR_SOFTNESS_ORTHO_ANG, PL_SLDR_DAMPING_DIR_LINEAR, PL_SLDR_DAMPING_DIR_ANGULAR, PL_SLDR_DAMPING_LIMIT_LIN, PL_SLDR_DAMPING_ORTHO_LIN, PL_SLDR_DAMPING_ORTHO_ANG, PL_SLDR_RESTITUTION_DIR_LINEAR, PL_SLDR_RESTITUTION_DIR_ANGULAR, PL_SLDR_RESTITUTION_LIMIT_LIN, PL_SLDR_RESTITUTION_ORTHO_LIN, PL_SLDR_RESTITUTION_ORTHO_ANG, PL_SLDR_MAX_ANG_MOTOR_FORCE, PL_SLDR_MAX_LIN_MOTOR_FORCE, PL_SLDR_TARGET_LIN_MOTOR_VEL, PL_SLDR_TARGET_ANG_MOTOR_VEL, PL_HC2_LOWER_LIMIT, PL_HC2_UPPER_LIMIT, PL_P2P_UPDATE_RHS,

value

El valor asignado.

Errores:

PL_INVALID_NAME es generado si la restriccion no existe o el id es incorrecto.

PL_INVALID_ENUM es generado si param es una constante invalida o no permitida.
