Documentacion de la API
plShape3f

Poner un valor a un parametro especifico

void plShape3f(Plenum param, Plfloat x, Plfloat y, Plfloat z)

Parametros:

param

Especifica el parametro para introducir el valor. Puede ser PL_EXTENT, PL_PLANE_NORMAL, PL_LOCAL_SCALING.

x y z

Los valores asignados.

Notas:

Las siguientes constantes son aceptadas por param:

PL_EXTENT poner la extension. x y z debe ser el vector(x,y,z).

PL_PLANE_NORMAL asignar la direccion de colision de un plano estatico. x y z debe ser el vector(x,y,z) del normal.

PL_LOCAL_SCALING escalar una forma de colision. x y z debe ser el vector(x,y,z) de escala.

Errores:

PL_INVALID_OPERATION es generado en raras ocaciones.

PL_INVALID_ENUM es generado si param es una constante invalida o no permitida.
